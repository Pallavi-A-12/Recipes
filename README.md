# 🍽️ Odin Recipes

A simple recipe website built using **HTML** as part of **The Odin Project Foundations Course**.

This project focuses on strengthening core HTML skills such as page structure, links, images, lists, and file organization.

---

## 📌 Project Overview

The **Odin Recipes** website contains a main homepage with links to multiple recipe pages.

Each recipe page includes:

- Recipe title
- Recipe image
- Description of the dish
- Ingredients list
- Step-by-step cooking instructions
- Navigation back to the homepage

This project is intentionally simple and focuses only on **HTML fundamentals**. Styling with CSS will be added in future learning stages.

---

## 🚀 Features

✅ Homepage with links to multiple recipes  
✅ Separate recipe pages  
✅ Internal page navigation using relative links  
✅ Recipe descriptions and cooking steps  
✅ Images for each dish  
✅ Proper HTML document structure
